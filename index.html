Game directory

Point to the folder containing the game's main executable. The scanner detects upscaler DLL entry points from this location.

What the scanner looks for: nvngx_dlss.dll, sl.dlss.dll, libxess.dll, amd_fidelityfx_dx12.dll, amd_fidelityfx_upscaler_dx12.dll, amd_fidelityfx_framegeneration_dx12.dll, ffx_fsr2/fsr3 variants, and their Linux equivalents. At least one must be present.
Scanning game files

Analyzing the directory for upscaler entry points. Files with alternate naming conventions (e.g. Clair Obscur, Space Marine 2) are also detected.

Installation options

Configure what gets installed. Defaults are tuned for your selected GPU. Any existing files will be backed up before being replaced.

OptiScaler version
DXGI Spoofing
Installs OptiScaler as dxgi.dll — enables GPU spoofing for non-Nvidia hardware. Required for AMD and Intel to unlock the DLSS input path.
FakeNvapi
Installs nvapi64.dll alongside OptiScaler. Enables Anti-Lag 2 on AMD RDNA1+ and LatencyFlex for Reflex-enabled games. Do not use with online games.
Vulkan Spoofing
Enables Vulkan GPU spoofing in OptiScaler.ini. Needed for Vulkan-based DLSS games on non-Nvidia hardware. Not compatible with all games (Doom Eternal still doesn't work).
dlssg-to-fsr3 (Nukem's mod)
Enables FSR3 frame generation for games with native DLSS-FG. Bundled since OptiScaler 0.9. DX12 only.
Generate OptiScaler.ini
Writes a pre-configured OptiScaler.ini based on your GPU. If one already exists it will be backed up first.
Overlay shortcut key
The key that opens the in-game overlay. Default is INSERT. Some compact/laptop keyboards don't have it — pick an alternative or press any key.
Press any key on your keyboard to set it as the shortcut (modifiers like Shift/Ctrl are ignored).
